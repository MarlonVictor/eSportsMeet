## Sobre
eSportsMeet é uma aplicação web com o intuito de unir jogadores de diversos jogos. Você pode usá-lo para encontrar um duo ou até mesmo um squad 
completo para jogar o seu game favorito.  

Essa aplicação foi construída durante as aulas do Ignite lab 03 patrocinado pela Rocketseat.

## Tecnologias Utilizadas
* Typescript
* React
* TailwindCSS
* Radix UI
* Prisma
* NodeJS
* Express

## Como Rodar

### Clone o repositório
```bash
$ git clone https://github.com/MarlonVictor/eSportsMeet.git
```

### Executar o Servidor
```bash
# Vá para a pasta do servidor 
$ cd eSportsMeet/server

# Instalar Dependências
$ yarn install

# Executar Aplicação 
$ yarn start
```

### Executar projeto Web
```bash
# Vá para a pasta da web 
$ cd eSportsMeet/web

# Instalar Dependências
$ yarn install

# Executar Aplicação 
$ yarn start
```
